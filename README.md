# CardiacDeux

A personal TeuxDeux-style daily planner with two-way Notion sync.

## Setup

1. Go to [console.anthropic.com](https://console.anthropic.com/settings/keys) and create an API key
2. Open the app and paste your key — it's stored only in your browser
3. Hit **Sync Notion** to pull your tasks

## Features

- Weekly column view (Mon–Sun)
- Tasks from your Notion Task Database auto-populate by due date
- Check tasks done here → updates Notion automatically
- Overdue task banner
- Someday section
- Works on all devices via any browser

## Notion Database

Connected to your **Task Database** — pulls tasks with a Due Date set and Status not "Done".
Category colors: Personal (blue), Cardiac (pink), Estate (amber), Property Management (green), Benefits Board (orange), Growth (teal), Research (gray), Acquisitions (purple).
